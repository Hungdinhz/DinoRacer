# Tài nguyên game

Đặt các file sau để sử dụng sprites và âm thanh:

## images/
- `dino.png` - Khủng long đứng/chạy (44x47 px)
- `dino_duck.png` - Khủng long cúi (44x28 px)
- `cactus.png` - Xương rồng (34x70 hoặc 34x100 px)
- `bird.png` - Chim (46x40 px)

## sounds/
- `jump.wav` - Âm thanh nhảy
- `gameover.wav` - Âm thanh game over
- `score.wav` - Âm thanh mốc 100 điểm

Nếu không có file, game vẫn chạy bình thường với đồ họa vẽ tay.
